# Developer Portfolio - Cloud Deployment Project

Proyek ini adalah implementasi deployment aplikasi full-stack ke Cloud dengan praktik terbaik DevOps (CI/CD, Monitoring, dan Security).

## 🔧 Fitur Teknis
- **Deployment:** Berhasil dideploy di Vercel Cloud Platform.
- **CI/CD Pipeline:** Otomatisasi Build & Deploy menggunakan GitHub Actions terintegrasi Vercel.
- **Monitoring:** Menggunakan Vercel Analytics & Speed Insights untuk melacak performa.
- **Keamanan:** Penyimpanan Secret Variable menggunakan Vercel Environment Variables.
